环境版本及其他说明¶

语言环境版本说明¶

Python 版本：3.6.5Sphinx 版本： 1.8.1

Python 版本是 3.X.X 版本即可，不要求具体版本必须相同。 Sphinx 由于安装较早，版本相对旧一些，新版可能略有差别（新版配置时减掉了一些选项），在具体使用和相关操作上没有影响。

示例说明¶

若无特殊说明，则 Windows 和 MacOS 系统所做操作相同。 操作不同的地方，基本集中在安装 Python 和 Sphinx 安装部分，会单独说明。
